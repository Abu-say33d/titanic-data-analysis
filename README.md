
Titanic Data Analysis

This project analyzes and cleans the Titanic dataset using Python and Pandas.

 Steps Performed
1. Loaded the dataset from CSV
2. Displayed data structure and summary statistics
3. Handled missing values:
   - Filled missing `Age` with mean
   - Filled missing `Embarked` with mode
   - Dropped the `Cabin` column
4. Verified all missing values handled successfully

 Tools Used
- Python
- Pandas
- Jupyter Notebook
